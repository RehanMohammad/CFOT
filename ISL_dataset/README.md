\# ISL Skeleton Dataset



This folder contains the ISL Skeleton Dataset released with the "Cross-Frame Optimal Transport for Skeleton-Based Hand Gesture Recognition" paper.



\## Contents

\- ISL\_dataset.zip : Full dataset (train/val/test splits)



\## Description

\- 50 isolated ISL signs

\- 3 subjects

\- 10 repetitions per sign

\- Skeletons extracted using MediaPipe



\## Usage

Unzip the dataset:

```bash

unzip ISL\_dataset.zip
